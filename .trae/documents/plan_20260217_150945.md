## 二、待优化项（按优先级）

### P0: CLI 多语言选项暴露
**问题**: 刚添加的多语言支持未暴露到 CLI
**改进**:
1. 在 CLI `wiki` 命令添加 `--wiki-languages <langs>` 选项
2. 更新 `WikiConfig` 类型添加 `wikiLanguages` 字段
3. 更新 `DEFAULT_CONFIG` 包含默认语言配置

### P1: 共享路径对齐 Qoder 风格
**问题**: Qoder 使用 `.qoder/repowiki`，我们使用可配置路径
**改进**:
1. 默认共享路径改为 `.tsdgen/wiki`（保持品牌一致性）
2. 支持多语言目录结构（如 `.tsdgen/wiki/zh/`、`.tsdgen/wiki/en/`）

### P2: Git 目录变更导入
**问题**: Qoder 支持从 Git 目录直接导入 Markdown 变更
**改进**:
1. 添加 `wiki import` 命令
2. 实现 `WikiSharingService.importFromGit()` 方法
3. 支持检测 Git 目录中的 Markdown 变更

### P3: 文档完善
**改进**:
1. 更新 README.md 添加多语言使用说明
2. 添加 CLI 选项文档

## 三、TDD 开发流程

每个改进项遵循：
1. **Red** - 先写失败的测试
2. **Green** - 实现最小代码使测试通过
3. **Refactor** - 重构优化
4. **Commit** - 提交代码
